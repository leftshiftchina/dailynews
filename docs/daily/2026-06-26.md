## 📚 AI学术论文

📏 **Ask, Don't Judge: Binary Questions for Interpretable LLM Evaluation and Self-Improvement**  
链接：https://arxiv.org/pdf/2606.27226v1  
概况：论文提出 BINEVAL，将开放式生成评估拆解为原子级二元问题，提升 LLM 评测的可解释性、多维度诊断能力和自我改进反馈质量。

🛡️ **Paved with True Intents: Intent-Aware Training Improves LLM Safety Classification Across Training Regimes**  
链接：https://arxiv.org/pdf/2606.27210v1  
概况：论文引入 AIMS 数据集，将用户意图作为安全分类的显式中间信号，提升监督微调、偏好学习、推理蒸馏和强化学习等训练范式下的安全识别能力。

🎥 **HarmVideoBench: Benchmarking Harmful Video Understanding in Large Multimodal Models**  
链接：https://arxiv.org/pdf/2606.27187v1  
概况：论文构建面向大型多模态模型的有害视频理解基准，覆盖隐含危害、深层上下文和解释性理由，推动视频内容安全评估从二分类走向细粒度理解。

✂️ **TOPS: First-Principles Visual Token Pruning via Constructing Token Optimal Preservation Sets for Efficient MLLM Inference**  
链接：https://arxiv.org/pdf/2606.27161v1  
概况：论文提出视觉 Token 剪枝方法 TOPS，通过构造最优保留集合降低多模态大模型推理开销，在效率与任务相关视觉信息保留之间取得平衡。

🤸 **Learning to Fold: prizewinning solution at LeHome Challenge 2026**  
链接：https://arxiv.org/pdf/2606.27163v1  
概况：论文介绍 LeHome Challenge 2026 双臂衣物折叠获奖方案，使用强化学习循环改进视觉-语言-动作策略，展示 VLA 模型在机器人精细操作中的落地潜力。
