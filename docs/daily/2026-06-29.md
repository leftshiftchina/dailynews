## 📚 AI学术论文

🧪 **Agentic Hardware Design as Repository-Level Code Evolution**
链接：https://arxiv.org/pdf/2606.28279v1
概况：论文提出 HORIZON 自演化智能体框架，将硬件设计视为仓库级代码演进任务，通过执行评估器、验收谓词和 Git 工作树循环实现自动化硬件设计迭代。

🛡️ **Agent-Native Immune System: Architecture, Taxonomy, and Engineering**
链接：https://arxiv.org/pdf/2606.28270v1
概况：研究面向自主智能体提出“原生免疫系统”架构，针对记忆投毒、工具链操控和多智能体协议攻击等运行时风险建立防御分类与工程方案。

👁️ **Vision-Default, Prior-Override: Causal Mechanisms of Perception-Knowledge Conflict in Vision-Language Models**
链接：https://arxiv.org/pdf/2606.28273v1
概况：论文通过激活修补、组件消融和机制分析，研究视觉语言模型在视觉证据与记忆知识冲突时的内部因果机制，有助于提升多模态系统可靠性。

🛰️ **RSICCLLM: A Multimodal Large Language Model for Remote Sensing Image Change Captioning**
链接：https://arxiv.org/pdf/2606.28266v1
概况：论文提出面向遥感图像变化描述的多模态大语言模型，解决双时相遥感图像细粒度变化理解和数据稀缺下的大模型适配问题。

🗣️ **HPRO: Hierarchical Progressive Reward Optimization via Preference Extraction for Emotional Text-to-Speech**
链接：https://arxiv.org/pdf/2606.28249v1
概况：研究针对 LLM 式语音合成中的情感表达不足，提出分层渐进奖励优化方法，通过偏好提取改善情感 TTS 的韵律和表现力。

## 💻 AI开源项目

⚙️ **StanleyLi-Buging/loop-flow - 多智能体自动化开发引擎**
链接：https://github.com/StanleyLi-Buging/loop-flow
概况：采用 Director 规划、Worker 执行、Reviewer 审查的多智能体协作模式，可在 DeepSeek、GPT、Claude、Kimi 等模型间切换，适合搭建轻量 AI 开发团队。

🧰 **renanflustosa/octo-cluster - 本地 AI 辅助开发工作台**
链接：https://github.com/renanflustosa/octo-cluster
概况：集成 RAG、记忆、Token 经济和验证门禁能力，面向 IDE 无关的本地 AI 辅助研发流程，可扩展阶段命令和能力包。

🔎 **Arshu30/Local-RAG-Document-Search - 本地 RAG 文档搜索引擎**
链接：https://github.com/Arshu30/Local-RAG-Document-Search
概况：基于 FAISS、sentence-transformers 和 google-genai SDK 构建本地向量检索与上下文问答，适合私有文档检索和轻量知识库问答。

🖥️ **kobolingfeng/deepseek-desktop - DeepSeek 原生桌面客户端与编码智能体**
链接：https://github.com/kobolingfeng/deepseek-desktop
概况：面向 DeepSeek 及 OpenAI 兼容 API，提供流式对话、文件与命令工具、内置终端、Office 文档处理、MCP 和子智能体能力，采用 C++/WebView2 与 React 架构。

🔁 **NickAiNYC/infini - 智能体可移植开放标准**
